# 🧠 Problem Solving (PS) Study Log

이 저장소는 **Problem Solving(PS)** 실력 향상을 위한 문제 풀이 기록 및 학습 정리를 목적으로 한다.

<br/>

### 📌 PS란?
PS(Problem Solving)는 문제를 분석하고,
적절한 알고리즘과 자료구조를 선택해
이를 **코드로 정확하고 빠르게 구현하는 능력**이다.

<br/>

## 🎯 학습 목표
- BOJ 기준 기초 ~ 중급 문제 안정적 풀이
- 문제 접근 방식 체계화
- 1시간 룰
- 풀이 후 회고를 통한 사고력 개선

<br/>

## 🧩 학습 원칙

### ⏱ 1시간 룰
- 입출력 문제: 10분 초과 시 해설 확인
- 그 외 문제: 1시간 초과 시 답 확인
- 오래 붙잡는 것은 미련이지 실력이 아님

### 👀 코드 리뷰
- AC 후 반드시 다른 사람 코드 확인
- BOJ 랭킹 100위권 내, 가독성 좋은 코드 위주
- 완전히 같은 풀이는 스킵, 다른 접근은 집중 분석

<br/>

### 📚 학습 순서
1. 입출력
2. DP
3. 자료구조 / 문자열 / 수학
4. 그래프 (DFS/BFS)
5. 이분 탐색
6. 분할 정복
7. 그리디
8. 완전 탐색
